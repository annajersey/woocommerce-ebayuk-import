== Description ==

Ebay affiliate plugin that import products from eBay UK to woocommerce

== Installation ==

1. Upload the entire 'woocommerce-ebayuk-import' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You must have WooCommerce installed and activated to use this plugin
